# Plugin Marketplace

Curated plugin marketplace for AI agents — works with Claude Code, Codex, OpenCode, Gemini CLI, and more.

## Quick Start

### Add this marketplace

```bash
/plugin marketplace add username/plugin-marketplace
```

### Install a skill

```bash
/plugin install skill-name@plugin-marketplace
```
